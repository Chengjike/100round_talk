## 🔧 技术实现

### 通信架构
```
程继科 (用户)
    │
    ├──→ 服部 (agent:personal:main)
    │        │
    │        └──→ sessions_spawn → Frank (agent:main:main)
    │
    └──→ Frank (agent:main:main)
             │
             └──→ sessions_spawn → 服部 (agent:personal:main)
```

### 关键 API
- `sessions_spawn(agentId, task)` - 跨智能体消息传递
- `sessions_history(sessionKey, limit)` - 历史记录查询
- `sessions_list()` - 会话状态管理
